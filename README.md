📊 OLA Booking Data Analysis (SQL + Power BI)
🚀 Project Overview

This project focuses on analyzing OLA ride booking data using SQL and Power BI to extract meaningful insights and support data-driven decision-making.

The goal of this project is to understand booking patterns, customer behavior, cancellations, and overall business performance.

🛠️ Tools & Technologies
SQL (MySQL)
Power BI
Excel (for initial data handling)
📁 Dataset Description

The dataset contains information related to:

Booking ID
Customer ID
Vehicle Type
Booking Status
Ride Distance
Booking Value
Payment Method
Driver & Customer Ratings
Cancellation Reasons
📌 Key Objectives
Analyze successful and canceled bookings
Identify top customers and ride patterns
Evaluate driver and customer ratings
Understand cancellation reasons
Calculate revenue and performance metrics
🧠 SQL Analysis Performed
Created database and structured data
Built SQL views for:
Successful bookings
Average ride distance by vehicle type
Top 5 customers by number of rides
Analyzed:
Customer & driver cancellations
Payment methods (e.g., UPI)
Ratings (driver & customer)
Total booking value
Incomplete rides with reasons
📊 Power BI Dashboard

Developed an interactive dashboard to visualize key insights:

🔹 KPIs Included:
Total Bookings
Successful vs Canceled Rides
Total Revenue
Average Ride Distance
Customer & Driver Ratings
🔹 Features:
Slicers (Vehicle Type, Payment Method, Booking Status)
Interactive charts (bar, pie, KPI cards)
Easy-to-understand visual insights
🔍 Key Insights
Identified peak booking trends
Found major reasons for ride cancellations
Observed differences in performance across vehicle types
Highlighted impact of cancellations on revenue
